# cletereid.com

Source for https://cletereid.com

A hand-built static website. No framework. No build step.

## Stack
- HTML
- CSS
- JavaScript

## Structure
- Individual HTML files per page
- Shared styles in `styles.css`
- Minimal JavaScript in `script.js`
- Static assets in `/images`
- SEO and platform files (`sitemap.xml`, `site.webmanifest`, icons)
- Apache config via `.htaccess`

## Local use
Open `index.html` directly in a browser, or serve the directory with any static file server.

## Deployment
Deployed as static files on a standard web host. No compilation or tooling required.
